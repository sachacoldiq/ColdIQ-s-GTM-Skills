# Claygent Guide

## What is Claygent?

AI research agent that combines Google Search, ChatGPT, and web scraping into a single tool. It browses the web, finds public data, and reports results.

## How to Access

1. Open a table
2. Go to the Enrichment panel (right side)
3. Select "Claygent" under the AI section

## Model Options

- **Claygent Neon:** Clay's flagship, optimized for extraction/formatting into columns
- **GPT-4:** For deeper reasoning
- **Claude Opus:** For deeper reasoning

## Credit Cost

- Simple questions (yes/no): 1 credit
- Deeper analyses: 2-3 credits

## Output Formatting

Available formats: text, number, URL, true/false, custom

## Example Prompt

```
Visit this company's homepage. Does it offer a free trial?
Return 'Yes' if they mention free trial, demo, or trial period.
```

## When to Use Claygent

- Data doesn't exist in databases
- Need real-time context
- Very specific/nuanced information needed

## Claygent vs AI Integrations

- **Claygent** = autonomous web search, scraping, extraction
- **AI Integrations** (OpenAI, Claude, etc.) = text generation, analysis of existing data

## Best Practices

- Ask Claygent to specify the source of information
- Specify the source platform (Google, LinkedIn) in the prompt
- Use built-in templates for common tasks
- Iterate without spending credits in the builder

## AI Integrations Available

- OpenAI/GPT
- Google Gemini
- Anthropic Claude
- Tavus (video generation)

**Use cases:** Text generation, Image creation, Data analysis, Personalized messaging
